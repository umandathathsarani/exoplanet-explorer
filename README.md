## 📖 About The Project

**Exoplanet Explorer** is an educational platform designed to bring the wonders of astrobiology and astrophysics to the general public. Built with modern web technologies, it features a deeply immersive, dark-themed space interface.

Users can dive into our comprehensive **10-Chapter Astronomy Textbook**, test their knowledge in our **Interactive Quiz System**, and eventually search live exoplanet databases.

### ✨ Key Features

- 📚 **Comprehensive Textbook:** 10 detailed chapters covering everything from basic definitions to the search for extraterrestrial life and next-generation space telescopes.
- 🧠 **Interactive Quizzes:** Test your knowledge with Easy, Medium, Hard, and Bonus challenges. Includes real-time scoring and answer reveals.
- 🎨 **Immersive UI:** A stunning, cinematic dark theme powered by TailwindCSS with micro-animations and glowing interactive elements.
- 📱 **Fully Responsive:** Optimized for both desktop observatories and mobile devices.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need Node.js and npm installed on your local machine.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/umandathathsarani/exoplanet-explorer.git
   ```
2. Navigate to the frontend directory
   ```sh
   cd exoplanet-explorer/frontend
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```
